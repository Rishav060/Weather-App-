# 🌦️ Weather App

A simple and clean weather application that shows real-time weather data using an API.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature, humidity, and wind speed
* ☁️ Dynamic weather icons
* ⚡ Fast and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Weather API

## 📂 Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
│── Assets/
```

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
